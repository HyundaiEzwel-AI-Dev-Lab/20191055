# HPMS 테스트 계정 (목업)

로컬 개발·화면 확인용 계정입니다. 데이터 출처: `src/data/mockUsers.js`

## 로그인 페이지 (PAG-M-COM-02)

| 항목 | 내용 |
|---|---|
| 화면 ID | PAG-M-COM-02 |
| URL | `/login` (루트 `/` 접속 시 로그인으로 리다이렉트) |
| 구현 파일 | `src/views/LoginView.vue` |
| figma 참고 | `01_로그인.html` |

### 화면 구성

- **좌측**: 시스템 소개 문구 + 프로젝트 현황 KPI (전체 27 / 접수 6 / 진행 14 / 완료 6)
- **우측**: 아이디 · 비밀번호 입력, ID 저장, 비밀번호 재설정, 로그인 버튼

### 입력·기능

| 기능 | 설명 |
|---|---|
| 아이디 | 사번 또는 외주 웹메일 ID (`placeholder`: USER ID) |
| 비밀번호 | 표시/숨김 토글 가능 |
| ID 저장 | 체크 시 `localStorage` (`hpms.savedLoginId`)에 아이디 저장 |
| 비밀번호 재설정 | POP-M-COM-03 팝업 (`PasswordResetModal.vue`) |
| 로그인 성공 | `/inbox`(내업무)로 이동 |

### 로그인 검증 순서 (목업)

1. ID/PW 미입력 → `ID 또는 PW를 입력해주세요`
2. 미등록 계정 → `등록되지 않은 계정입니다.`
3. 잠금/휴직/퇴직 계정 → 상태별 안내 메시지
4. 비밀번호 5회 초과 → `로그인 가능 횟수를 초과했습니다...`
5. 비밀번호 불일치 → `ID 또는 PW가 일치하지 않습니다.` (실패 횟수 +1)
6. 성공 → 내업무 이동
