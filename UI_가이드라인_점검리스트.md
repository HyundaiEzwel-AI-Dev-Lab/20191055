# UI/UX 가이드라인 대조 점검 리스트

> 기준 문서: [사내시스템_웹_UIUX_가이드라인_규칙.md](./사내시스템_웹_UIUX_가이드라인_규칙.md)
> 작성일: 2026-07-16
> 성격: 코드 조사 결과 리스트업 (아직 반영/수정 안 됨). 체크박스는 조치 완료 여부 체크용.

---

## 공통 (전역 UI / 모든 화면에 영향)

- [ ] **[필수]** 사이트맵 없음 — 전체 메뉴를 계층화해 보여주는 사이트맵 기능 없음 (`AppSidebar`는 로그인 사용자의 LNB만 노출)
- [ ] **[필수]** 통합검색 "검색" 텍스트 레이블 미노출 — `AppHeader.vue` 검색 버튼이 돋보기 아이콘 + `title` 툴팁만 있고 상시 노출되는 텍스트 레이블 없음
- [ ] **[필수]** 로고/시스템명 좌측 상단 상시 노출 위반 — 사이드바를 접으면(`is-collapsed`) 로고("H")와 브랜드명("PMS")이 `display:none`으로 완전히 사라짐
- [ ] **[필수]** 텍스트 명도대비 4.5:1 미달 — `--lnb-muted:#9aa0a8` on 흰 배경 대비 약 2.6:1 (사이드바 그룹라벨, 비활성 탭 라벨 등 다수 사용)
- [ ] **[필수]** 데이터 조회 결과 정렬 기능 전무 — 검토한 **22개 화면 전체**에서 테이블 헤더 클릭 정렬이 하나도 구현되어 있지 않음
- [ ] **[필수]** 미저장 상태 페이지 이동 시 확인 요청 없음 — `beforeRouteLeave`/`beforeunload` 가드가 앱 전체에 0건. `BaseModal`도 배경 클릭/×버튼 클릭 시 dirty 체크 없이 즉시 닫힘 (ProjectInfoView의 "취소" 버튼 1곳만 예외)
- [ ] **[권장]** 단축키(F2/F3/F6/F7/F9) 미구현 — 통합검색의 F2(전역)만 있고 나머지 전무, 툴팁 안내도 없음
- [ ] **[필수]** 도움말/관리자 문의 기능 없음 — 기능별 도움말, 문의 링크가 앱 어디에도 없음
- [ ] **[권장]** 탭이 가로 스크롤 구조 — 전역탭/서브탭 모두 `overflow-x:auto` — 가이드는 "Tab: 가로 스크롤 불가"를 명시 (페이지 전체 가로 스크롤은 없어 그 부분은 준수)
- [ ] **[권장]** 모달 중첩 — `PasswordResetModal`이 열린 상태에서 그 위에 별도 alert/confirm 오버레이가 추가로 뜸

## 로그인 (LoginView.vue)

- [ ] **[필수]** 로그인 오류 메시지가 계정 존재 여부를 노출 — 미등록 계정은 "등록되지 않은 계정입니다", 비밀번호 오류는 "ID 또는 PW가 일치하지 않습니다"로 서로 다르게 안내 (가이드는 보안상 동일하게 안내 권고)

## DefectView (결함관리)

- [ ] **[필수]** 일괄 선택 기능 없음 — 건별 처리(`DefectDetailModal`)만 가능
- [ ] **[권장]** 기간 필터 없음 — 최근 N일 등 기본조건 제공 안 됨
- [ ] **[필수]** `DefectDetailModal` 닫기 시 미저장 확인 없음

## RequirementView (요구사항 관리)

- [ ] **[권장]** 기본 조회기간이 `2026-01-01~01-31`로 하드코딩(최근기간 아님), 필터 접힘 상태로 기본 미노출
- [ ] **[필수]** 페이지네이션 아이콘버튼(«‹›») 4개 툴팁 없음
- [ ] **[필수]** `RequirementFormModal` placeholder로만 작성안내 제공(상하단 설명 텍스트 없음)
- [ ] **[필수]** 필드별 글자수 카운터 불일치 (비고만 카운터 있음)
- [ ] **[필수]** `RequirementBulkRegisterModal` 대량등록 파일서식/크기 안내 텍스트 없음

## TestCaseView (테스트 라이브러리)

- [ ] **[권장]** 다른 화면과 달리 Enter 검색만 유일하게 지원(인터랙션 불일치)
- [ ] **[권장]** 기간 필터 없음

## TestRunView (테스트 수행)

- [ ] **[필수]** "조회" 버튼에 클릭 핸들러 없음 — 죽은 버튼
- [ ] **[필수]** 페이지네이션 자체가 없음(전체 렌더링)
- [ ] **[권장]** 기간 필터 입력값은 있으나 화면에 렌더링 안 됨(죽은 필드)
- [ ] **[필수]** 결과 변경 일괄 처리 기능 없음

## ScenarioView (시나리오 관리)

- [ ] **[필수]** 펼침 화살표(▶▼) 버튼 툴팁 없음
- [ ] **[권장]** 기간 필터 없음
- [ ] **[필수]** 일괄 선택 없음
- [ ] **[필수]** `ScenarioBulkRegisterModal` 파일서식/크기 안내 없음

## WbsView (WBS 관리)

- [ ] **[필수]** 페이지네이션 없음(전체 렌더링)
- [ ] **[필수]** `WbsBulkScheduleModal` 사유 입력이 placeholder로만 안내
- [ ] **[필수]** "작업제외" 처리 시 재확인창 없음

## CommonCodeView (공통코드 관리)

- [ ] **[필수]** 등록 모달 필수입력 시각 구분 없음
- [ ] **[필수]** 정렬순서 값 범위/단위 안내 없음
- [ ] **[권장]** 분류 내 키워드 검색 없음

## UserMgmtView (사용자 관리)

- [ ] **[필수]** 필수/선택 필드 시각 구분 없음
- [ ] **[필수]** 엑셀 추출 기능 자체 없음

- [ ] **[권장]** 승인대기 건이 상단 정렬되지 않음

- [ ] **[필수]** 이슈 등록 textarea가 태그(@) 사용법을 placeholder로만 안내
- [ ] **[필수]** 담당자 20명 제한이 사전 안내 없이 사후 alert로만 통보
- [ ] **[필수]** 담당자/업무범주 일괄 등록 없음
- [ ] **[필수]** 칩(chip) 삭제 버튼 툴팁 없음
- [ ] **[필수]** `hasWbs=false`일 때 담당자/업무범주 삭제가 재확인 없이 즉시 실행
- [ ] **[필수]** 이슈/답글에 작성자·관리자 구분 스타일 없음

## ScenarioEditView (시나리오 편집)

- [ ] **[필수]** 케이스명/절차 필수입력 표시(*) 없음
- [ ] **[필수]** 절차 순서 변경(▲▼) 버튼 툴팁 없음
- [ ] **[필수]** 절차 삭제가 confirm 없이 즉시 실행
- [ ] **[권장]** 절차/예상결과 입력칸이 표 셀 내 단일 라인이라 장문 입력에 불충분
